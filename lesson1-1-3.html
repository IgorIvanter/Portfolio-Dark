First phrases

Simple sentences and questions

It's time to move on to the more conversational part of learning German – making sentences. To do this, we will need some new words. It is these that we will use in the first stages to describe the objects that are around us. First we will describe our room and everything in it, and then gradually go beyond it and, thus, expand our worldview. In this section we will not make complex constructions, but we will start with the simplest question Was ist das?, which means What is this?

Was?
What?

ist
is

Das
this

Answering such a question, we can give objects names.

Look at the examples.

Das ist ein Bleistift
This is a pencil

Das ist ein Tisch
This is a table

The noun in such sentences should be accompanied by an indefinite article ein or eine, which indicates that we give the subject a name, just like in English.

Das ist ein Handy.
This is a mobile phone

Das ist ein Bett
This is a bed

Try to translate the following sentences:

What's it? — It's a chair

What's it? — This is the window

What's it? — It's a flower

What's it? — It's a pen

What's it? — It's a TV

Questions

Now let's look how the sentence:

This is a pencil

Can be turned into a questiion:

Is this a pencil?

To do that, we need to change the word order, just like in the english sentence.

Ist das ein Bleistift?
Is this a pencil?

Now ask questions to the following sentences:

This is a boy

This is a dog

This is a flower

This is a chair

This is a city

This is a teacher

This is a handbag

Answering a question with Ja

To answer such a question, you can say Ja and add an affirmative sentence.

Of course, each of these questions can be answered with just Ja. – without adding anything further. You will be understood. However, do not forget that you need to train your speech and get used to the language. To do this, try to speak in full sentences for now. It is easier to shorten a phrase than to say a longer construction correctly.

Answer the following questions in the affirmative:

Is this a table?

Is this a man?

Is this a pencil?

Answering a question with Nein

Sometimes we need to give a negative answer to a question. To do that, you can say Nein, which means No and replace ein/eine with kein/keine, which is a word that is used to negate sentences with nouns.

Ist das eine Tür? Nein, das ist keine Tür.

Is this a door? No, this is not a door.

Ist das ein Mann? Nein, das ist kein Mann.

Is this a man? No, this is not a man.

Ist das ein Bleistift? Nein, das ist kein Bleistift. Das ist ein Kuli.

Is this a pencil? No, this is not a pencil. This is a pen.

In this lesson, we learned how to build direct questions. Direct questions are questions that require a direct answer yes or no. We have learned to give both affirmative and negative answers to such questions. The constructions turned out to be quite simple, but they form the basis of the language. You'll see how very soon a simple sentence Das ist ein Tisch, by replacing various words and adding new ones, will grow like a big tree and turn into a voluminous and beautiful Spanish language.
